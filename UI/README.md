# UI 使用指南

此文件夹包含本地多模态 AI 智能助手的图形用户界面 (GUI)。

## 环境要求

- Python 3.10+
- PyQt6

## 如何运行

1. 打开终端（PowerShell 或 CMD）。
2. 激活 conda 环境：
   ```bash
   conda activate exp2
   ```
3. 导航到项目根目录：
   ```bash
   cd d:\fan\Project\PycharmProject\DuoMoTai\Experiment2
   ```
4. 运行 UI 脚本：
   ```bash
   python UI/main_ui.py
   ```
